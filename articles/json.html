Javascript JSON parser

2011-02-14

evalJSON is a full JSON parser implemented in Javascript.
It checks passed string according the defined JSON lexical grammar by ECMA-262-5.

The semantic of evalJSON is exactly as the semantic defined by ECMA-262-5 for JSON.parse built-in function.

evalJSON should not be used as a fallback implementation of JSON.parse.
It is better to be used for debugging and testing purposes of your JSON strings.

If the passed JSON string is not syntactical correct, evalJSON throws a SyntaxError with appropriate message.
Also it reports the line and the column number of the error.

It covers all test cases from my JSON.parse compatibility table.
