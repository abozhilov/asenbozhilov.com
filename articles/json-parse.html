JSON.parse compatibility table

2010-11-17

ECMA-262-5 describes JSON lexical grammar in 15.12.1.1 The JSON Lexical Grammar section.
Conformance implementation must follow these lexical grammar rules in their JSON.parse implementation.

In the test cases, when I have used Unicode Escape Sequences (\u HexDigit HexDigit HexDigit HexDigit) I suppose to be inserted character represented by this sequence instead of UnicodeEscapeSequence itself.
